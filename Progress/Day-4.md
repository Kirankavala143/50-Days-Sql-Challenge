Day 4: Data Cleaning (Handling Inconsistent Text Values)
Objective

Improve data quality by identifying and standardizing inconsistent text values across multiple tables to ensure accurate analysis and reporting.

Tasks Completed
Identified inconsistent text formats and naming variations in columns such as city and department names.
Standardized text values using string functions such as UPPER(), LOWER(), and text formatting techniques.
Removed unnecessary leading and trailing spaces using TRIM().
Corrected spelling variations and inconsistent naming conventions.
Validated distinct values before and after cleaning to ensure consistency.
Key Learning
Text inconsistencies can significantly impact filtering, grouping, and reporting.
Values that appear similar to humans may be treated as different records by SQL.
Standardization is a critical step in any data-cleaning workflow.
Functions like TRIM(), UPPER(), and LOWER() are essential for maintaining data quality.
Outcome

The dataset now contains standardized and consistent text values, improving the reliability of analysis, aggregation, and business insights.
