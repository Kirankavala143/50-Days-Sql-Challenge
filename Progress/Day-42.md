# Day 42 – SQL Data Cleaning

## Overview
This project focuses on cleaning raw restaurant order data using SQL. The objective is to prepare the data for accurate analysis by removing inconsistencies and improving data quality.

## Tasks Performed

- Created duplicate (cleaned) tables from raw tables
- Removed leading and trailing spaces using TRIM()
- Standardized text using UPPER()
- Replaced NULL values with meaningful defaults
- Removed duplicate records using ROW_NUMBER()
- Verified cleaned data after transformation

## SQL Concepts Used

- CREATE TABLE AS SELECT
- UPDATE
- TRIM()
- UPPER()
- ROW_NUMBER()
- DELETE
- WHERE
- Window Functions

## Tables

- cleaned_customers
- cleaned_orders
- cleaned_restaurants
- cleaned_menu_items
- cleaned_order_details

## Learning Outcome

This project helped me understand how SQL is used in real-world data cleaning before performing data analysis and visualization.
