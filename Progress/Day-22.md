📌 Day 22 – SQL Window Functions | Department Analysis
📖 Objective

Learned how to use SQL Window Functions to perform department-level analysis while preserving individual employee records.

🔍 Tasks Completed

✅ Calculated the average salary of each department for every employee.

✅ Calculated the total salary of each department using SUM() Window Function.

✅ Calculated the average performance rating of each department using employee performance data.

This project helped me understand how Window Functions allow us to calculate aggregate values without grouping rows, making department-level analytics much easier while keeping employee-level details intact.
