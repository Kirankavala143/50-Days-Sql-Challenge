# Day 11 - SQL Joins & Aggregations

## Tasks Completed

1. What is the employee name, department, and performance rating?
2. What is the complete profile of each employee (department, salary, performance)?
3. How many salary records does each employee have?
4. What is the total salary paid to each employee?

## Concepts Practiced

- INNER JOIN
- LEFT JOIN
- Aggregate Functions
  - COUNT()
  - SUM()
- GROUP BY
- Multi-table Analysis

## Key Learning

Data is often distributed across multiple tables. SQL JOINs help connect related information, while aggregate functions help generate meaningful insights from raw data.
