# Day 16 - SQL Challenge

## Topic
Aggregation Functions & GROUP BY

## Dataset
Employee Salary Dataset

## Queries Solved

### 1. Total Salary Per Employee
Used SUM() to calculate total salary paid to each employee.

### 2. Average Salary Per Employee
Used AVG() to calculate average salary of each employee.

### 3. Count Salary Records Per Employee
Used COUNT() to find number of salary entries for each employee.

### 4. Maximum Salary Per Employee
Used MAX() to identify highest salary received by each employee.

## Concepts Learned
- GROUP BY
- SUM()
- AVG()
- COUNT()
- MAX()
- JOIN operations
- Employee salary analysis

## Key Takeaway
Aggregation functions help transform raw data into meaningful business insights.
