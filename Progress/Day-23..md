# Day 23 – SQL Window Functions (LAG)

## 📌 Objective
Learn and implement the LAG() window function to compare current and previous records for analytical reporting.

---

## ✅ Tasks Completed

### 1. Current Salary vs Previous Salary
Displayed each employee's current salary along with their previous salary using the LAG() window function.

### 2. Salary Difference Analysis
Calculated the difference between the current salary and the previous salary for every employee.

### 3. Attendance Trend Analysis
Compared the current attendance status with the previous attendance status to identify whether attendance improved, declined, or remained unchanged.

Reviewed and rectified data quality issues across all tables and columns before performing the analysis.

## 🎯 Key Learning

The LAG() function makes it easy to compare the current row with the previous row without using self joins. It is useful for trend analysis, salary comparisons, attendance monitoring, and other time-based analytical reports.
