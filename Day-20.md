# Day 20 - SQL Window Functions (ROW_NUMBER)

## Topics Covered
- ROW_NUMBER()
- PARTITION BY
- ORDER BY
- Ranking Records
- Top-N Analysis

## Tasks Completed

### 1. Latest Salary per Employee
Retrieve the most recent salary record for each employee.

### 2. First Salary per Employee
Retrieve the first salary record for each employee.

### 3. Rank Salary Records
Assign rankings to salary records employee-wise.

### 4. Top 2 Salary Records per Employee
Retrieve the top two salary entries for every employee.

## Key Learnings
- ROW_NUMBER() assigns unique rankings to rows.
- PARTITION BY creates groups.
- ORDER BY controls ranking order.
- Useful for analytics and reporting queries.

## SQL Concepts Used
- Window Functions
- ROW_NUMBER()
- PARTITION BY
- ORDER BY
- Subqueries

## Outcome
Learned how to perform ranking and Top-N analysis using SQL Window Functions.
