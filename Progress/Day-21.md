# Day 21 - SQL Window Functions (RANK & DENSE_RANK)

## Tasks Completed

1. Ranked employees based on salary using `RANK()`.
2. Ranked employees within each department using `DENSE_RANK()`.
3. Calculated average performance rating and ranked employees.
4. Retrieved the Top 3 highest-paid employees.

## Concepts Practiced

- Window Functions
  - RANK()
  - DENSE_RANK()
- OVER()
- PARTITION BY
- ORDER BY
- Average Calculation
- Employee Ranking
- Top N Records

## Key Learning

Window functions allow ranking and analysis without grouping data. They help compare rows, rank employees within departments, and retrieve top-performing records while preserving all rows.
