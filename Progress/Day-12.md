# Day 12 - SQL Aggregations and Grouping

## Tasks Completed

1. Find the average salary in each department
2. Calculate attendance count for each employee
3. Identify employees belonging to the same department
4. Find employees having multiple salary records

## Concepts Practiced

- AVG()
- COUNT()
- GROUP BY
- HAVING
- GROUP_CONCAT()
- JOINS

## Key Learning

Aggregation functions help summarize large datasets and generate meaningful business insights. GROUP BY and HAVING are powerful tools for identifying patterns, duplicates, and trends within data.
